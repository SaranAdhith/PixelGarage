# PixelGarage

The official landing page for **PixelGarage** — a software studio that designs and builds
**iOS & Android apps, custom websites and backends** for companies, from scratch to launch.

Built as a fast, dependency‑free static site with a retro **pixel‑art / Pac‑Man** theme.

## ✨ Highlights

- **Drawable hero canvas** — a 16×16 pixel sprite you can paint with the colour palette.
- **Arcade animations** — a marquee, a Pac‑Man‑and‑ghosts chase strip, and floating chips.
- **The crew** — hover any sprite in the gallery to reveal that team member's mini‑profile.
- **Founder profiles** — the co‑founders rendered as pixel Pac‑Man & ghost characters.
- **Fully responsive** and respects `prefers-reduced-motion`.

## 🚀 Run it locally

No build step — it's plain HTML/CSS/JS. Either:

```bash
# open directly
open index.html          # macOS
xdg-open index.html      # Linux

# …or serve it
python3 -m http.server 8000
# then visit http://localhost:8000
```

## 🎨 Tech
